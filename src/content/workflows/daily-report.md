# Daily Report Generator

Automate your daily reporting with data from multiple sources.

## Use Cases

- Daily sales summary reports
- Operations metrics dashboards
- Team productivity tracking
- Financial daily close reports

## How It Works

1. Scheduled trigger runs at your desired time
2. Pull data from databases and spreadsheets
3. Calculate metrics and trends
4. Format into readable report
5. Distribute via Slack and email

## Setup Time

2-3 hours

## Customization Options

- Multiple data sources
- Custom metrics and KPIs
- Various output formats (PDF, HTML, Slack blocks)
- Different schedules per report type
- Conditional alerts for anomalies
