## Tentang Project

Aplikasi manajemen tugas yang memungkinkan tim untuk berkolaborasi secara realtime dengan fitur drag-and-drop intuitif.

## Fitur Utama

- **Kanban Board** - Tampilan board dengan drag-and-drop
- **Realtime Sync** - Update otomatis untuk semua anggota tim
- **Notifikasi** - Push notification untuk deadline dan assignment
- **Filter & Search** - Pencarian dan filter task yang powerful

## Tech Stack

- **Frontend**: React, React DnD, Tailwind CSS
- **Backend**: Firebase (Firestore, Auth, Cloud Functions)
- **Realtime**: Firebase Realtime Database

## Pembelajaran

Project ini mengajarkan pentingnya optimistic UI updates dan conflict resolution dalam aplikasi realtime.
