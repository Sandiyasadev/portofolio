## Tentang Project

Tool yang membantu developer membuat portfolio profesional dengan cepat, mengambil data langsung dari profil GitHub mereka.

## Fitur Utama

- **GitHub Integration** - Ambil repo, contributions, dan profile otomatis
- **Template Themes** - Berbagai tema yang bisa dikustomisasi
- **SEO Optimized** - Meta tags dan sitemap otomatis
- **Deploy Ready** - One-click deploy ke Vercel/Netlify

## Tech Stack

- **Framework**: Astro
- **API**: GitHub REST API, GraphQL
- **Language**: TypeScript
- **Styling**: CSS Variables, Responsive Design

## Highlight

Project ini menggunakan Astro untuk static site generation, menghasilkan website yang sangat cepat dengan Lighthouse score 100.
